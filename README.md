# 🎮Doodle Jump Game

## ℹAbout:
This is a desktop game, a clone of the doodle jump mobile game.

## 💻To deploy localy:
```
1)git clone https://github.com/iFoxtrot33/Doodle-Jump.git
2)cd Doodle-Jump
3)make install
```
